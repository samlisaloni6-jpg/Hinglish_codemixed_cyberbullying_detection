# Hinglish Cyberbullying Detection using DistilBERT

This project detects cyberbullying in Hinglish code-mixed text using DistilBERT.

## Project Overview
Cyberbullying in multilingual and code-mixed languages like Hinglish is difficult to detect using traditional NLP methods. This project uses DistilBERT for binary classification.

## Features
- Text preprocessing
- Duplicate removal
- DistilBERT fine-tuning
- Custom sentence prediction
- Performance evaluation

## Dataset
Hinglish cyberbullying dataset (25000 samples)

## Model Used
DistilBERT Multilingual

## Results
- Accuracy: 87.5%
- Precision: 0.90
- Recall: 0.88
- F1-score: 0.87
